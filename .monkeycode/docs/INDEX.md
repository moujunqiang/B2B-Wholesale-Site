# B2B Wholesale Site - 项目文档索引

## 项目概述

**项目名称**: B2B Wholesale Site  
**技术栈**: Cloudflare Workers + Hono + D1 + R2  
**项目类型**: 外贸 B2B 批发独立站  

## 核心功能

### 前台功能
- **产品目录**: 展示产品列表和详情
- **产品分类**: 多级分类浏览
- **询盘系统**: 客户在线发送询盘
- **多语言支持**: 可扩展的多语言翻译系统

### 后台管理
- **产品管理**: 添加、编辑、删除产品
- **分类管理**: 管理产品分类树
- **询盘管理**: 查看和回复客户询盘
- **网站设置**: 配置站点参数
- **简单认证**: HTTP Basic Authentication

```
│   用户浏览器     │────▶│  Cloudflare CDN │

                    ┌────────────▼────────────┐
                    │   Cloudflare Workers    │
                    │   (Hono 框架 + API)     │
                    └────────────┬────────────┘
                                 │
              ┌──────────────────┼──────────────────┐
              │                  │                  │
     ┌────────▼────────┐ ┌──────▼───────┐ ┌───────▼────────┐

```
